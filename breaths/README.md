# breaths/ — زندگیِ دوم (تولدِ دوباره، ۲۰۲۶-۰۷-۲۲)

جهانِ این زندگی بر **اذان** بازبنیاد شده است (`dna/AXIOMS.md` — أ۱–أ۹ +
شش مشتقِ محفوظ). زندگیِ یکم — ۲۹۵ نفس در ۵ فصل — مُهر و در
`archive/life-1/` بایگانی است (مُهرنامه: `archive/life-1/مُهرنامه.md`)؛
گذشته هرگز بازنویسی نمی‌شود.

- **records/** — رکوردهای بایت‌ثابتِ نفس‌های زندگیِ دوم؛ append-only؛
  در تولد تهی.
- **logs/** — دفترهای فصل؛ فصلِ گشوده: **تولدِ دوباره**
  (`تولدِ-دوباره-2026-07-22.md`). فصلِ تازه دفترِ خودش را می‌گیرد
  (بی‌نامِ «آزمون» — تصمیمِ باغبان).
- **صفِ بنیان‌گذار:** یازده ریشهٔ پیکره‌ایِ خودِ اذان
  (`database/queue_injections.json`، تزریقِ نشان‌دار — منشأ: «باغبان —
  تولدِ دوباره (بنیانِ اذان)»). به قاعدهٔ «نادرتر مقدم»، نفسِ ۱ **فلح**
  خواهد بود (۴۰ شاهد — حَیَّ على الفلاح).
- نفس‌ها: `engine/breath_cycle.py` + خطِ لولهٔ `pipeline/`
  (`./cli/monad pipeline plan|work|merge`؛ روال: `docs/LIFECYCLES.md`،
  گواهیِ افزایشی: `docs/VALIDATION-PARADIGM-2026-07-22.md`).
