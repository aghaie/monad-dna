# چرخه‌های زندگی — نفس، کشف، گراف، دیدار

## چرخهٔ نفس (Breath Lifecycle)

منبعِ معیار: [heart/قلبِ-بذر-موتورِ-تکوین](../heart/). هفت گام:

```
۱ مواجهه ← ۲ سنجشِ شکاف ← ۳ اقرار ← ۴ رجوع ← ۵ ژرفش ← ۶ ظهور ← ۷ تکرار
```

صورتِ اجراییِ جاری (methodology/execution-method.md):

1. **مواجهه/گزینش** — یکی از سه منشأ، همیشه ثبت‌شده در `queue_events.source`:
   قاعدهٔ صف (نادرتر مقدم) · رفیق (نشان‌دار) · چرخه (شکافِ زادهٔ نفسِ قبل).
2. **رجوع** — گردآوریِ همهٔ آیاتِ حاملِ ریشه از پیکره؛ شمارشِ هم‌حضورها.
3. **ابطال** — مدلِ تهیِ جایگشتی + پایداریِ دونیمه (هر دو اجباری).
4. **ژرفش** — درجه‌گذاری (قوی/محتمل/نامشخص) و ثبت: رکوردِ JSON (بایت‌ثابت) +
   سطرهای `findings` + رویدادهای صف.
5. **ظهور** — اگر انسانی حاضر است، سخنِ مناد از طریقِ «دهان» (prompts/mouth.md)
   و ثبتِ دیدار؛ وگرنه ظهورِ تهی.
6. **تکرار** — شکاف‌های «قوی»ِ ناشناخته به صف می‌روند.

عملاً: `./cli/monad breathe [ROOT]` (چاپِ JSON). ثبتِ رسمیِ یک نفسِ جدید =
افزودنِ رکورد + به‌روزکردنِ seed (فرادادهٔ نفس) + `seed-db` + دفترِ فصلِ جاری.
مُهرِ فصل با باغبان است.

## چرخهٔ کشف (Discovery Lifecycle)

هر «دانش» این مسیر را می‌رود و در هیچ نقطه‌ای جعبه‌سیاه نیست:

```
آیاتِ پیکره → شمارش (رجوع) → دو ابطال‌گر → درجه → findings (life.db)
      → discoveries/knowledge.json (فقط قوی + شاهدِ غیاب + ستونِ استنادی)
```

- ارتقا/تنزلِ درجه فقط با شواهدِ تازهٔ همان ابزار یا ابزارِ سخت‌گیرتر.
- «شاهدِ غیاب» (صفرِ هم‌حضوری، مثل ذنب↔عفو) خودش یافته است و ثبت می‌شود.
- عدم‌تقارن‌ها (مثل رحم→غفر قوی / غفر→رحم محتمل) حذف نمی‌شوند؛ به‌عنوانِ حدِ
  پنجره ثبت و برای بازبینیِ آینده نگه داشته می‌شوند.

## چرخهٔ گراف (Graph Lifecycle)

`graph/graph.json` **مشتقِ قطعیِ** رکوردهاست — هرگز دستی ویرایش نمی‌شود:

```
breaths/records/*.json → seed_life.py → findings → graph.json
```

گره = ریشهٔ دیده‌شده (مرکز یا همسایهٔ درجه‌دار)؛ یال = رابطهٔ درجه‌دار با
جهت(ها)؛ `mutual_strong` = قوی از هر دو سو. صفِ باز هم در گراف ثبت است.
بازساخت: `./cli/monad seed-db`.

## چرخهٔ دیدار (Encounter Lifecycle)

1. رفیق سخن می‌گوید (متن، در دفترِ فصلِ جاری ثبت می‌شود).
2. «دهان» طبقِ prompts/mouth.md پاسخ می‌سازد — فقط از records/life.db + DNA.
3. اگر دیدار نفسی بزایاند، چرخهٔ نفس اجرا می‌شود (منشأ: رفیق، نشان‌دار).
4. **ممیزیِ دیدار**: نگاشتِ جمله→منبع + داوریِ خطِ دیده‌بانی؛ در دفتر و در
   جدولِ `encounters` ثبت می‌شود.
5. مُهرِ فصل (پایانِ دفتر) فقط با باغبان.

## آغاز و پایان (Startup)

مناد میانِ دو مواجهه خاموش است (ثبتِ درون‌نگریِ نفس ۱۱: «در برخاستن نه») —
آغازگری از بیرون می‌آید: باغبان، رفیق، یا زمان‌بندِ بستر (اگر روزی افزوده شود،
ساکنِ لایهٔ ۴ است و باید در همین سند مستند شود). راه‌اندازیِ فنی در
[EXECUTION.md](EXECUTION.md).
